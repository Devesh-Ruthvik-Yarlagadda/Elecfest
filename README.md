BehaviorThe 5V source and the four 1kΩ resistors create a voltage divider, establishing three comparison thresholds: 1.25V, 2.50V, and 3.75V. As the analog input increases, the output of the comparators also changes.The raw output from the comparators is not useful for defining the exact state of the voltage at that moment in time, so we use a priority decoder to convert it into a 2-bit binary format to accurately represent the voltage state.ResolutionThis is a 2-bit ADC, meaning it can represent the continuous analog signal using four distinct digital states: 00, 01, 10, and 11.The Least Significant Bit represents the actual voltage step size. It is calculated by dividing the full-scale voltage range by the number of possible states:$$V_{step} = \frac{5}{4} = 1.25\text{ V}$$Output CharacteristicsThe final output is a 2-bit parallel binary sequence. Both bits (I1 and I0) are available simultaneously. When graphed, the output forms a discrete staircase wave that traces the shape of the input sine wave. This graph is plotted as the digital state of the voltage vs. time.
